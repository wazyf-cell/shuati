# 题目导入格式规范

本系统支持 TXT 和 Excel 两种导入方式。以下分别说明。

---

## TXT 格式（推荐）

### 基本结构

```
[题型标签]
题干内容
A. 选项A
B. 选项B
C. 选项C
答案：正确选项
解析：解析内容（可选）
```

- 每道题之间用**一个空行**隔开
- `[题型标签]` 必须顶格写，前面不能有任何字符（包括空格、编号）

> ⚠️ **重要：标签必须使用半角方括号 `[]`**，不要用 `【】` `「」` 等其他括号。
> 例如写 `[判断]`，不要写 `【判断题】`。

### 题型标签

| 标签 | 含义 | 选项要求 |
|------|------|---------|
| `[单选]` | 单选题 | 必须提供 A/B/C/D 等选项行 |
| `[多选]` | 多选题 | 必须提供 A/B/C/D 等选项行 |
| `[判断]` | 判断题 | **必须**提供 `A. 正确` / `B. 错误` 两行 |
| `[填空]` | 填空题 | 不需要选项 |
| `[简答]` | 简答题 | 不需要选项 |
| `[简答-大题]` | 含多小题的大题（阅读理解、完型填空等） | 不需要选项 |

### 选项格式（单选/多选/判断）

字母 + 分隔符（`.` `：` `:` `、` 均可）+ 空格 + 选项内容

合法写法：

```
A. 北京
A：北京
A: 北京
A、北京
```

**判断题必须写：**
```
A. 正确
B. 错误
```

### 答案格式

- 答案行**必须**以 `答案：` 或 `答案:` 开头，`正确答案：` 不会被识别
- **单选**填一个字母：`答案：A`
- **多选**连续写：`答案：AB` 或 `答案：A,B` 或 `答案：A、B`
- **判断题**推荐用字母：`答案：A`（正确）`答案：B`（错误），也支持中文 `答案：对` / `答案：正确` / `答案：错` / `答案：错误`
  - 不要写 `正确答案：错`（前缀不对），判断题的 `正确`/`错误` 指判断结果而非选项字母
- **填空**答案用逗号或顿号分隔多个空：`答案：H2O, CO2`
- **简答**直接写答案文本：`答案：光合作用是...`
- **简答-大题**见下方独立说明

### 解析（可选）

以 `解析：` 或 `解析:` 开头，`知识点：` 不会被识别（会被忽略，不会报错）

### 简答-大题格式（完型/阅读/材料题通用）

大题题干后，逐行列出小题和答案。**选项写在小题内容里：**

```
小题编号：题目内容（含选项）
答案编号：答案内容
```

**编号格式：**

| 格式 | 示例 |
|------|------|
| `小题N：` | `小题1：简述现象` |
| `答案N：` | `答案1：温室效应` |

**选项写在一行**（用双空格分隔，刷题显示时会按选项格式展示）：

`小题1：1. 题目 __.  A. 选项A  B. 选项B  C. 选项C  D. 选项D`

### 完整示例

```
[单选]
以下哪个是中国的首都？
A. 北京
B. 上海
C. 广州
D. 深圳
答案：A
解析：北京是中国的首都。

[多选]
以下哪些是哺乳动物？
A. 猫
B. 狗
C. 蛇
D. 鸟
答案：AB
解析：猫和狗是哺乳动物。

[判断]
太阳从东边升起。
A. 正确
B. 错误
答案：A
解析：这是基本常识。

[填空]
水的化学式是______，二氧化碳的化学式是______。
答案：H2O, CO2
解析：常见化学式

[简答]
请简述光合作用的主要过程。
答案：光合作用是植物利用光能、水和二氧化碳合成有机物并释放氧气的过程。
解析：光合作用的基本概念

[简答-大题]
阅读以下材料，回答问题。
小题1：材料中描述的现象是什么？
答案1：温室效应
小题2：该现象产生的主要原因是什么？
答案2：二氧化碳排放过多
解析：环境保护相关知识点

[简答-大题]
阅读下面短文，从每题所给选项中选出最佳选项。
When I was a child, I loved to visit my grandmother's farm.
小题1：1. The author visited the farm __.  A. every weekend  B. every summer  C. every winter  D. every month
答案1：B
小题2：2. The author learned __ from his grandmother.  A. how to cook  B. how to read  C. how to be patient  D. how to drive
答案2：C
小题3：3. What is the best title?  A. My Grandmother's Farm  B. How to Work Hard  C. Summer Vacation  D. My Parents
答案3：A
解析：文章讲述作者在奶奶农场的经历
```

### 常见错误

| 错误写法 | 问题 | 正确写法 |
|---------|------|---------|
| `1. [单选]` | 标签前有编号 | `[单选]` |
| ` [单选]` | 标签前有空格 | 删掉前导空格 |
| `【判断题】` | 标签用了书名号+多"题"字 | `[判断]` |
| `[判断]` 后没写选项 | 判断也必须写 A/B 选项行 | 加上 `A. 正确` / `B. 错误` |
| `正确答案：错` | 前缀必须是 `答案：`，不能是 `正确答案：` | `答案：B` 或 `答案：错` |
| `知识点：xxx` | 解析器不认识，被忽略 | `解析：xxx` |
| 两题间没有空行 | 解析器可能将两题视为一题 | 加一个空行 |
| `A 北京`（选项无分隔符） | 无法识别选项 | `A. 北京` |

## TXT 格式：给 AI 的 Prompt

如果你用其他 AI 帮忙整理题目，直接贴这段：

> 请将以下题目严格按此格式整理：
>
> ```
> [单选]
> 题干
> A. 选项
> B. 选项
> C. 选项
> D. 选项
> 答案：A
> 解析：解析内容
> ```
>
> **规则：**
> 1. `[单选]` / `[多选]` / `[判断]` / `[填空]` / `[简答]` / `[简答-大题]` 顶格写，前面不要任何编号或空格
> 2. 选项用 `A. ` / `B. ` / `C. ` / `D. ` 格式（仅单选/多选/判断需要；**判断必须写 `A. 正确` / `B. 错误`**）
> 3. 答案用 `答案：` 开头（不要写 `正确答案：`）
>    - 单选/多选：填字母，如 `答案：A` 或 `答案：AB`
>    - 判断：`答案：A`（正确）/ `答案：B`（错误），也支持 `答案：对` / `答案：正确` / `答案：错` / `答案：错误`
>    - 填空：多个空用逗号分隔，如 `答案：H2O, CO2`
>    - 简答：直接写答案文本
>    - 简答-大题：用 `小题1：` / `答案1：` 逐小题列出
> 4. 解析可选，用 `解析：` 开头（不要用 `知识点：`）
> 5. 每道题之间空一行

## Excel 格式

第一行为表头，支持以下列名（中英文均可）：

| 列含义 | 支持的表头写法 |
|--------|--------------|
| 题型 | `题型` / `type` |
| 题干 | `题干` / `content` / `question` |
| 选项 A 到 F | `选项A` / `A` … `选项F` / `F` |
| 正确答案 | `正确答案` / `correctAnswer` / `answer` |
| 解析 | `解析` / `analysis` / `explanation` |

### 题型填写

| 填写内容 | 含义 |
|---------|------|
| `单选` / `single` / `单选题` | 单选题 |
| `多选` / `multiple` / `多选题` | 多选题 |
| `判断` / `judge` / `判断题` | 判断题 |
| `填空` / `fill` / `填空题` | 填空题 |
| `简答` / `short` / `简答题` | 简答题 |

### 答案填写

- **单选**：直接写字母（`A`）
- **多选**：`AB` / `A,B` / `A、B` 均可
- **判断**：`对` / `正确` / `1` / `true` 为正确，其余为错误
- **填空**：逗号或顿号分隔多个空（`H2O, CO2`）
- **简答**：直接写答案文本

### 示例

| 题型 | 题干 | 选项A | 选项B | 选项C | 选项D | 正确答案 | 解析 |
|------|------|-------|-------|-------|-------|---------|------|
| 单选 | 以下哪个是首都？ | 北京 | 上海 | 广州 | 深圳 | A | 北京是首都 |
| 多选 | 哪些是哺乳动物？ | 猫 | 狗 | 蛇 | 鸟 | AB | 猫和狗是哺乳动物 |
| 判断 | 太阳从东边升起 | 正确 | 错误 | | | 对 | 基本常识 |
| 填空 | 水的化学式是\_\_\_\_ | | | | | H2O, CO2 | 常见化学式 |
| 简答 | 简述光合作用 | | | | | 植物利用光能... | 基本概念 |
