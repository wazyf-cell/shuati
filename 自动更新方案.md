# Windows 程序内自动更新 — 实现方案

## 整体思路

```
前端点击"自动更新"
    ↓ 调 Tauri Rust 命令
Rust 用 reqwest 下载新 exe 到 %TEMP%
    ↓
Rust 生成 update.bat 批处理脚本
    ↓
Rust 执行批处理脚本（独立进程），然后退出 app
    ↓
批处理等 app 退出后，复制新 exe 覆盖旧 exe，启动新版
```

---

## 一、Rust 依赖（Cargo.toml）

```toml
[dependencies]
reqwest = { version = "0.12", features = ["blocking"] }
```

去掉 `blocking` 用 async 也行，但 Windows 批处理方案用 blocking 更简单。

## 二、Rust 命令（lib.rs）

打开 `src-tauri/src/lib.rs`，新增下面两个命令：

### 2.1 新增 import

```rust
use std::path::PathBuf;
use std::process::Command;
```

### 2.2 下载命令：`download_update`

```rust
#[tauri::command]
fn download_update(url: String) -> Result<String, String> {
    let response = reqwest::blocking::get(&url)
        .map_err(|e| format!("下载失败: {}", e))?;

    let bytes = response.bytes()
        .map_err(|e| format!("读取响应失败: {}", e))?;

    let temp_dir = std::env::temp_dir();
    let exe_path = temp_dir.join("shuati_update.exe");

    std::fs::write(&exe_path, &bytes)
        .map_err(|e| format!("写入文件失败: {}", e))?;

    exe_path.to_str().map(|s| s.to_string())
        .ok_or_else(|| "路径转换失败".to_string())
}
```

### 2.3 应用更新命令：`apply_update`

```rust
#[tauri::command]
fn apply_update(app_handle: tauri::AppHandle) -> Result<(), String> {
    let temp_dir = std::env::temp_dir();
    let new_exe = temp_dir.join("shuati_update.exe");
    let bat_path = temp_dir.join("update_shuati.bat");

    // 获取当前 exe 路径
    let current_exe = std::env::current_exe()
        .map_err(|e| format!("获取当前路径失败: {}", e))?;

    // 生成 update.bat 脚本
    let bat_content = format!(
        "@echo off\r\n\
         :loop\r\n\
         tasklist /FI \"IMAGENAME eq shuati.exe\" 2>NUL | find /I /N \"shuati.exe\" >NUL\r\n\
         if \"%ERRORLEVEL%\"==\"0\" (\r\n\
           timeout /t 1 /nobreak >NUL\r\n\
           goto loop\r\n\
         )\r\n\
         copy /Y \"{}\" \"{}\" >NUL\r\n\
         if errorlevel 1 (\r\n\
           echo Update failed > \"%TEMP%\\update_error.txt\"\r\n\
           exit /b 1\r\n\
         )\r\n\
         start \"\" \"{}\"\r\n\
         del \"%~f0\"\r\n",
        new_exe.display(),
        current_exe.display(),
        current_exe.display()
    );

    std::fs::write(&bat_path, bat_content)
        .map_err(|e| format!("写入批处理文件失败: {}", e))?;

    // 执行批处理（不等待）
    Command::new("cmd")
        .args(&["/c", bat_path.to_str().unwrap()])
        .spawn()
        .map_err(|e| format!("启动更新脚本失败: {}", e))?;

    app_handle.exit(0);
    Ok(())
}
```

### 2.4 注册命令

在 `lib.rs` 的 `run()` 函数中注册：

```rust
pub fn run() {
    tauri::Builder::default()
        .plugin(tauri_plugin_log::Builder::new().build())
        .plugin(tauri_plugin_dialog::init())
        .plugin(tauri_plugin_fs::init())
        .invoke_handler(tauri::generate_handler![
            download_update,
            apply_update,
            // ... 其他已有命令 ...
        ])
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

## 三、前端调用（Settings/index.tsx）

检测到 Tauri 环境时调 Rust 命令，否则退化到浏览器下载：

```typescript
// 判断是否 Tauri 环境
const isTauri = typeof window !== 'undefined' && (window as any).__TAURI_INTERNALS__;

const handleDownload = useCallback(async () => {
  if (!updateInfo) return;

  // Tauri 桌面端：程序内自动更新
  if (isTauri) {
    try {
      const { invoke } = await import('@tauri-apps/api/core');
      addToast('正在下载更新...', 'info');
      const tempPath = await invoke('download_update', { url: updateInfo.downloadUrl });
      addToast('下载完成，正在更新...', 'info');
      await invoke('apply_update');
      // 到这步 app 会自动退出
    } catch (e: any) {
      addToast(`更新失败: ${e}`, 'error');
    }
    return;
  }

  // Android 或 Web：浏览器下载
  const url = isAndroid && updateInfo.apkDownloadUrl
    ? updateInfo.apkDownloadUrl
    : updateInfo.downloadUrl;
  if (url) {
    window.open(url, '_blank');
  }
  setShowUpdatePopup(false);
}, [updateInfo, isAndroid, addToast]);
```

## 四、用户感受

```
点"自动更新"
    ↓
Toast: "正在下载更新..."（几秒）
    ↓
Toast: "下载完成，正在更新..."
    ↓
app 自动关闭
    ↓
1-2 秒延迟（批处理等待旧进程退出）
    ↓
app 自动重新启动，已是新版
```

全程无弹窗、无浏览器、无手动操作。

## 五、注意事项

1. **reqwest 的 blocking 功能**需要 `tokio`，Tauri 默认带的 tokio 已满足，不需要额外配置
2. **路径包含空格**：上面已经用双引号包裹，没问题
3. **Windows 安全软件**：`copy /Y` 覆盖 exe 可能会触发杀软提示，这是正常的
4. **更新失败处理**：如果 `copy` 失败，会在 `%TEMP%\update_error.txt` 写错误信息
5. **测试方法**：在 `apply_update` 的 `Command::new("cmd")` 前加一行 `dbg!(&bat_content);` 查看生成的脚本内容
