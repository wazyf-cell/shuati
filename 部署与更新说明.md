# 部署与更新说明

> **Gitee 仓库**: `https://gitee.com/zhong-yongfu/shuati`
> **版本检查 URL**: `https://gitee.com/zhong-yongfu/shuati/raw/master/gitee-update/version.json`

## 一、项目结构

```
program6-shuati/
├── src-tauri/           ← Tauri 桌面端代码（Rust）
├── src/                 ← 前端代码（React + Vite）
├── gitee-update/        ← 更新文件目录
│   ├── version.json     ← 版本信息（改版本号时要更新）
│   ├── shuati.exe       ← 绿色版 exe
│   ├── shuati_*.exe     ← 安装包
│   └── app-debug.apk    ← Android APK
├── deploy.bat           ← 一键部署脚本
└── 部署与更新说明.md     ← 本文件
```

## 二、打包桌面应用

```bash
npm run tauri:build
```

产出物：

| 用途 | 文件 | 路径 |
|------|------|------|
| **绿色版（推荐分享）** | `shuati.exe` | `src-tauri/target/release/shuati.exe` |
| 安装包 | `shuati_0.1.0_x64-setup.exe` | `src-tauri/target/release/bundle/nsis/` |

## 三、发布新版（每次更新时）

### 完整步骤

**1. 改版本号**

- `src-tauri/tauri.conf.json` 中的 `version` 字段
- `src/components/Settings/index.tsx` 中的 `APP_VERSION` 常量

**2. 打包**

```bash
npm run tauri:build
```

**3. 更新 version.json**

编辑 `gitee-update/version.json`：

```json
{
  "version": "0.2.0",
  "downloadUrl": "https://gitee.com/zhong-yongfu/shuati/raw/master/gitee-update/shuati.exe",
  "apkDownloadUrl": "https://gitee.com/zhong-yongfu/shuati/raw/master/gitee-update/app-debug.apk",
  "notes": "新功能说明\\n换行用\\n"
}
```

**4. 一键部署**

```bash
双击 deploy.bat
```

自动完成：复制 exe/APK → 提交 Git → 推送到 Gitee

**或手动执行：**

```bash
copy src-tauri\target\release\shuati.exe gitee-update\
git add gitee-update/
git commit -m "update: v0.2.0"
git push origin master
```

## 四、自动更新流程

```
用户打开设置 → 自动检查更新
    ↓
请求 Gitee Raw → gitee-update/version.json
    ↓
对比 version ≠ APP_VERSION
    ↓
弹出通知 → 点"下载更新" → 下载新 exe/APK → 替换
    ↓
如果网络错误 → 显示手动下载链接
```

## 五、注意事项

- `version.json` 和 Settings 页面 `APP_VERSION` 必须一致
- 下载链接指向 Gitee Raw，国内直连，不需要梯子
- 导出数据包含 AI 配置（含 API Key），重置后不会包含
