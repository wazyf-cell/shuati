# 细化改进方案

## 一、收藏题库 UI —— 参考错题本

### 现状
收藏页只显示题目文本 + Flag 取消按钮，太简陋。

### 改为错题本风格

| 元素 | 来源 |
|------|------|
| 按题库分组折叠 | WrongBook 按题库分组逻辑 |
| 题目可折叠展开 | WrongBook 的 `collapsedQuestionIds` |
| 显示题型标签 | `typeBadges` + `typeLabels`（和错题本一致） |
| 显示选项 + 正确答案 | WrongBook 的 options 网格 |
| 显示错了几次 | `wrongQuestions` 查找 wrongCount |
| AI解析按钮 | 复用 `<AIExplanationCard>` |
| 移除收藏 | 错题本"移除"按钮换为"取消收藏" |
| 刷这些题 | 每组顶部"刷这些题"按钮 |

### 颜色方案
本项目主色是 **accent（蓝紫色）**，收藏页全部使用蓝色系（`bg-blue-*` / `from-blue-* to-blue-*`），和 Dashboard、WrongBook 统一。

### 文件

| 文件 | 改动 |
|------|------|
| `src/components/FavoriteBank/index.tsx` | 重写，参考 WrongBook |

## 三、导入弹窗居中

### 现状
ImportModal 的 `card` 内有 `mx-4` 且没有 overflow 控制，在窄屏时可能偏移。

### 修复
确保内层 div 完全居中：去掉 `mx-4`，`card` 内部用 `w-full` + `max-w-xl` + overflow-y-atuo。

| 文件 | 改动 |
|------|------|
| `src/components/Bank/ImportModal.tsx` | 调整居中 CSS |

## 四、题目编辑页 AI 解析按钮

### 改动
`QuestionForm.tsx` 的"题目解析" textarea 下方加"🤖 AI生成解析"按钮：
- 点击 → 调用 `generateExplanation(question, [])`（无用户答案）
- loading 时显示"AI 正在思考..."
- 结果自动填入 textarea
- 复用已有的 AI 逻辑（`src/utils/ai.ts`）

| 文件 | 改动 |
|------|------|
| `src/components/Bank/QuestionForm.tsx` | 加 AI 按钮 + 逻辑 |
