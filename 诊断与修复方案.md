# APK 下载源切换到 GitHub

## 原则

- **exe/setup** → 保持 Gitee（国内下载快）
- **APK** → GitHub Raw（Gitee 会拦截 APK 返回 zip）
- **version.json** → 保持 Gitee（更新检测不受影响）

## 改动

| 文件 | 改哪里 | 改动 |
|------|--------|------|
| `gitee-update/version.json` | `apkDownloadUrl` | Gitee → `https://raw.githubusercontent.com/wazyf-cell/shuati/main/gitee-update/app-debug.apk` |
| `src/components/Settings/index.tsx` | 手动下载 APK 链接 | Gitee → GitHub Raw |
| `src/App.tsx` | `handlePopupDownload` 中 APK 下载 | 已从 `version.json` 读，不用改 |

## 不需要改的

| URL | 保持 |
|-----|------|
| `version.json` 更新检测 | `gitee.com/zhong-yongfu/shuati/raw/master/gitee-update/version.json` |
| `downloadUrl` (exe) | `gitee.com/zhong-yongfu/shuati/raw/master/gitee-update/shuati.exe` |
| `src-tauri/src/lib.rs` | Gitee 不变 |
| `App.tsx` UPDATE_URL | Gitee 不变 |
