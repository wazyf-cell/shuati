# 数据持久化测试方案

> 按功能模块逐项验证输入输出、存储/读取、刷新后状态保持。

---

## 一、题库 (Banks)

### 1.1 创建题库

- [ ] 点击「导入数据」→ JSON 文件 → 题库成功创建
- [ ] 刷新页面 → 题库仍然存在
- [ ] localStorage `quiz_banks` 中存在正确数据

### 1.2 编辑题库

- [ ] 题库广场中点击编辑 → 修改题库名称 → 保存
- [ ] 刷新 → 名称已更新

### 1.3 删除题库

- [ ] 点击删除按钮（红色）→ 确认 → 题库消失
- [ ] 该题库下的题目不可再被刷到
- [ ] 错题本中该题库的错题卡片显示「题目已不存在」

### 1.4 导出数据

- [ ] 点击导出 → 下载 JSON 文件
- [ ] JSON 中包含 `banks`、`wrong`、`records`、`config` 四部分
- [ ] 删除题库 → 重新导入该 JSON → 题库恢复

---

## 二、刷题 (Practice)

### 2.1 普通刷题

- [ ] 选择题库 → 配置抽题数量 → 开始刷题
- [ ] 答对 → 下一题 → 答错 → 下一题
- [ ] 提交 → 进入 Review 页面
- [ ] 刷新页面 → 练习状态重置（回到题库广场，不保留未完成的状态）

### 2.2 按题型分配数量

- [ ] 开启「按题型设置题目数量」→ 分别设置各题型数量
- [ ] 开始刷题 → 题目按设置的顺序和数量出现
- [ ] 导航栏按题型分段显示

### 2.3 排除已刷题目

- [ ] 开启「排除已刷题目」→ 已答对的题目不再出现
- [ ] 关闭 → 所有题目可再次刷到

### 2.4 计时模式

- [ ] 开启计时（倒计时 1 分钟）→ 开始刷题
- [ ] 时间到 → 自动提交 → 未答题目计为错误

### 2.5 查看正确答案

- [ ] 配置中开启「支持查看正确答案」→ 刷题时出现 💡 查看答案
- [ ] 点击 → 高亮正确答案 → 选项禁用 → 跳到下一题
- [ ] 查看过的题不计正确率
- [ ] 关闭开关 → 按钮消失

### 2.6 AI 解析（刷题中）

- [ ] 已配置 AI 平台 + 开启「启用AI解析」→ 刷题中显示 🤖 AI解析 按钮
- [ ] 点击 → 生成解析 → 显示结果
- [ ] 关闭开关 → AI 按钮消失
- [ ] 未配置 AI → 开关置灰

---

## 三、错题本 (Wrong Book)

### 3.1 错题记录

- [ ] 刷题答错 → 该题自动进入错题本
- [ ] 再次答错同一题 → `wrongCount` 增加
- [ ] 刷新页面 → 错题仍在

### 3.2 错题本视图

- [ ] 「按时间」视图 → 时间倒序排列 → 题库子组 → 题目默认折叠
- [ ] 「按题库」视图 → 按题库分组 → 题目默认折叠
- [ ] 点击时间组标题 → 折叠/展开题库组
- [ ] 点击题库组标题 → 折叠/展开题目列表
- [ ] 点击题目摘要行 → 展开完整卡片

### 3.3 错题卡片展开内容

- [ ] 显示「你的答案是: X ❌ 错误 正确答案: Y」
- [ ] 选项标记 ❌/✅
- [ ] 错误次数
- [ ] 题目解析（analysis）
- [ ] AI 解析按钮（可生成）
- [ ] 重刷此题 / 移除按钮

### 3.4 错题操作

- [ ] 选中多题 → 重刷选中 → 跳转刷题配置
- [ ] 重刷全部 → 跳转刷题配置
- [ ] 移除此题 → 从错题本消失
- [ ] 清空某题库 → 该题库错题全部移除
- [ ] 清空全部 → 确认 → 错题本清空

### 3.5 重刷后自动移除

- [ ] 错题重刷答对 → 该题从错题本删除
- [ ] 错题重刷答错 → 留在错题本，wrongCount 更新

### 3.6 搜索和筛选

- [ ] 搜索框 → 按内容过滤错题
- [ ] 题型筛选 → 只显示该题型
- [ ] 搜索 + 筛选同时生效
- [ ] 空搜索 → 显示「没有找到匹配的错题」

---

## 四、练习统计 (Statistics)

### 4.1 数据记录

- [ ] 每次刷题提交后 → 统计页增加一条记录
- [ ] 正确数 / 总数 / 正确率 正确显示
- [ ] 用时正确记录
- [ ] 趋势图显示正确率趋势

### 4.2 题库过滤

- [ ] 选「全部题库」→ 显示所有记录汇总
- [ ] 选具体题库 → 只显示该题库记录

### 4.3 清空统计

- [ ] 选具体题库 → 「清空该题库统计」→ 确认 → 该题库记录删除，其他题库保留
- [ ] 选全部题库 → 「清空全部统计」→ 确认 → 所有记录删除
- [ ] 清空后统计页显示「还没有练习记录」

---

## 五、AI 配置 (AIConfig)

### 5.1 平台配置

- [ ] 选择平台 → 切换平台 → 显示对应域名和模型建议
- [ ] 填写 API Key → 保存 → toast「AI 配置已保存」
- [ ] 刷新页面 → 打开 AI 配置 → 上次填写的内容回填
- [ ] API Key 在 localStorage 中 Base64 编码（非明文）

### 5.2 模型输入

- [ ] 输入框有推荐列表（平台预设模型）
- [ ] 可手动输入任意模型 ID
- [ ] 保存后自定义模型名持久化

### 5.3 提示词模板

- [ ] 3 个模板切换生效
- [ ] 编辑模板内容 → 保存 → 刷新 → 内容保留
- [ ] 重置当前模板 → 恢复默认
- [ ] 刷题/Review 中使用选择的提示词

### 5.4 测试连接

- [ ] 「测试连接」→ 成功/失败结果展示（绿色/红色卡片）
- [ ] 「询问模型」→ 显示模型自我介绍

### 5.5 清空 AI 缓存

- [ ] 点击「清空 AI 缓存」→ toast 确认
- [ ] 再次访问错题/Review → AI 解析需重新生成

### 5.6 未配置时的行为

- [ ] 未配置 AI 时点击 AI 解析 → toast「请先在导航栏配置 AI 平台」
- [ ] 配置页面的「启用AI解析」开关置灰 + 提示

---

## 六、Review 页

### 6.1 Review 进入

- [ ] 提交答案后自动进入 Review 页
- [ ] 顶部显示总览条（正确数/总数 + 正确率 + 用时）

### 6.2 逐题回顾

- [ ] 左导航 → 按题型分段（勾选时） / 三列平铺（不勾选时）
- [ ] ✅ 绿色标记 / ❌ 红色标记
- [ ] 点击题号 → 右侧切到对应题目
- [ ] 显示「你的答案是: X ✅ 正确」或「❌ 错误 正确答案: Y」

### 6.3 AI 解析

- [ ] 错题显示 AI 解析按钮/结果
- [ ] 已缓存 → 自动显示
- [ ] 未缓存 → 点击生成
- [ ] 一键生成所有错题解析（最多 3 并发）

### 6.4 底部操作

- [ ] 「再来一次」→ 回到配置页
- [ ] 「返回配置」→ 回到配置页

### 6.5 错题重刷 Review

- [ ] 错题重刷完成后进入 Review
- [ ] 答对 → 自动移出错题本
- [ ] 导航为三列平铺（无题型分段）

---

## 七、多题库联合刷题

- [ ] 选择多个题库 → 设置各题库抽题数量
- [ ] 设置题型出现顺序
- [ ] 开始刷题 → 按设置的顺序和数量出题
- [ ] 提交 → Review 正常工作

---

## 八、数据导入导出循环

### 8.1 导出 → 导入

- [ ] 导出 JSON → 删除所有题库 → 导入同一个 JSON
- [ ] 题库完全恢复（名称、题目数量一致）
- [ ] 错题本数据恢复
- [ ] 统计记录恢复

### 8.2 导入覆盖

- [ ] 已有数据时导入新 JSON → 新题库追加，旧题库保留
- [ ] 同 ID 题库被覆盖更新

---

## 九、存储兼容性

- [ ] 删除 localStorage 所有数据 → 刷新 → 应用正常运行（空状态）
- [ ] 旧版数据（缺 `correctAnswer`、`firstWrongAt`）→ 自动补全，不报错
- [ ] 旧版 `UserConfig`（缺新字段）→ 自动补默认值
