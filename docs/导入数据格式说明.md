# 刷题助手 — 导入数据 JSON 格式说明

## 概述

导入功能支持通过 JSON 文件批量导入**题库、错题本、练习记录和配置**。导出功能生成的备份文件可直接重新导入。

---

## 完整 JSON 结构

```json
{
  "banks": [ /* 题库数组 */ ],
  "wrong": [ /* 错题数组 */ ],
  "records": [ /* 练习记录数组 */ ],
  "config": { /* 用户配置 */ }
}
```

四个字段均为可选，只导入需要的部分即可。例如只导入题库：

```json
{
  "banks": [ ... ]
}
```

---

## 1. 题库（banks）

每个题库是一个对象：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | ✅ | 唯一标识，建议用时间戳或 UUID |
| `name` | string | ✅ | 题库名称，如"计算机基础" |
| `questions` | Question[] | ✅ | 题目数组 |
| `createdAt` | number | ❌ | 创建时间戳（毫秒），不填自动生成 |
| `updatedAt` | number | ❌ | 更新时间戳 |

```json
{
  "id": "1717000000000",
  "name": "计算机基础",
  "questions": [ ... ],
  "createdAt": 1717000000000,
  "updatedAt": 1717000000000
}
```

---

## 2. 题目（Question）

支持 5 种题型：**单选、多选、判断、填空、简答**。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | ✅ | 唯一标识 |
| `type` | string | ✅ | 题型：`single` / `multiple` / `judge` / `fill` / `short` |
| `content` | string | ✅ | 题干内容 |
| `options` | Option[] | ✅ | 选项数组（填空/简答可为空数组 `[]`） |
| `correctAnswer` | string[] | ✅ | 正确答案 |
| `analysis` | string | ❌ | 答案解析，默认为空 |
| `subType` | string | ❌ | 仅简答题：`"single"` 普通 / `"group"` 分组 |
| `subQuestions` | SubQuestion[] | ❌ | 仅分组简答题 |

### 选项（Option）

| 字段 | 类型 | 说明 |
|------|------|------|
| `key` | string | 选项标号，如 `"A"` `"B"` |
| `content` | string | 选项内容 |

### 分组简答子题（SubQuestion）

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 子题 ID |
| `label` | string | 子题标签，如"第1小题" |
| `answer` | string | 参考答案 |

---

## 各题型示例

### 单选题（single）

```json
{
  "id": "q001",
  "type": "single",
  "content": "HTTP 协议的默认端口号是？",
  "options": [
    { "key": "A", "content": "21" },
    { "key": "B", "content": "80" },
    { "key": "C", "content": "443" },
    { "key": "D", "content": "8080" }
  ],
  "correctAnswer": ["B"],
  "analysis": "HTTP 默认端口为 80，HTTPS 为 443。"
}
```

### 多选题（multiple）

```json
{
  "id": "q002",
  "type": "multiple",
  "content": "以下哪些属于面向对象编程语言？",
  "options": [
    { "key": "A", "content": "Java" },
    { "key": "B", "content": "C" },
    { "key": "C", "content": "Python" },
    { "key": "D", "content": "Assembly" }
  ],
  "correctAnswer": ["A", "C"],
  "analysis": "Java 和 Python 支持面向对象，C 是面向过程，Assembly 是汇编语言。"
}
```

### 判断题（judge）

```json
{
  "id": "q003",
  "type": "judge",
  "content": "TCP 是无连接的传输协议。",
  "options": [
    { "key": "A", "content": "正确" },
    { "key": "B", "content": "错误" }
  ],
  "correctAnswer": ["B"],
  "analysis": "TCP 是面向连接的，UDP 才是无连接的。"
}
```

### 填空题（fill）

`correctAnswer` 数组顺序对应填空顺序。

```json
{
  "id": "q004",
  "type": "fill",
  "content": "OSI 七层模型中，第三层是____层，第四层是____层。",
  "options": [],
  "correctAnswer": ["网络", "传输"],
  "analysis": "OSI 七层：物理→数据链路→网络→传输→会话→表示→应用。"
}
```

### 普通简答题（short — single）

```json
{
  "id": "q005",
  "type": "short",
  "content": "简述 HTTP 和 HTTPS 的区别。",
  "options": [],
  "correctAnswer": ["HTTPS 在 HTTP 基础上加了 SSL/TLS 加密，使用 443 端口，HTTP 使用 80 端口。"],
  "analysis": "",
  "subType": "single"
}
```

### 分组简答题（short — group）

```json
{
  "id": "q006",
  "type": "short",
  "content": "阅读以下材料，回答问题。\n\n材料：……（题目材料）",
  "options": [],
  "correctAnswer": [],
  "analysis": "",
  "subType": "group",
  "subQuestions": [
    { "id": "sq001", "label": "(1)", "answer": "参考答案1" },
    { "id": "sq002", "label": "(2)", "answer": "参考答案2" },
    { "id": "sq003", "label": "(3)", "answer": "参考答案3" }
  ]
}
```

> **注意**：分组简答题的 `correctAnswer` 留空数组即可，答案存在 `subQuestions[].answer` 中。评判时对每个子题做**模糊匹配**（忽略标点、空格、大小写）。

---

## 3. 错题本（wrong）

可选，通常由系统自动生成。导入时会合并到现有错题数据。

| 字段 | 类型 | 说明 |
|------|------|------|
| `questionId` | string | 错题 ID（对应 Question.id） |
| `bankId` | string | 所属题库 ID |
| `wrongCount` | number | 累计错误次数 |
| `lastWrongAt` | number | 最近一次错误时间戳 |
| `userAnswers` | string[] | 用户最后一次的错误答案 |

```json
{
  "wrong": [
    {
      "questionId": "q001",
      "bankId": "bank001",
      "wrongCount": 3,
      "lastWrongAt": 1717000000000,
      "userAnswers": ["A"]
    }
  ]
}
```

---

## 4. 练习记录（records）

可选，每次刷题的历史记录。

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 记录 ID |
| `bankIds` | string[] | 涉及的题库 ID 列表 |
| `questionIds` | string[] | 本次练习的题目 ID 列表 |
| `answers` | object | `{ [questionId]: string[] \| Record<string,string> }` |
| `results` | object | `{ [questionId]: true \| false }` |
| `startTime` | number | 开始时间戳 |
| `endTime` | number | 结束时间戳 |
| `totalCount` | number | 题目总数 |
| `correctCount` | number | 答对数量 |

```json
{
  "records": [
    {
      "id": "r001",
      "bankIds": ["bank001"],
      "questionIds": ["q001", "q002", "q003"],
      "answers": {
        "q001": ["B"],
        "q002": ["A", "C"],
        "q003": ["B"]
      },
      "results": {
        "q001": true,
        "q002": true,
        "q003": true
      },
      "startTime": 1717000000000,
      "endTime": 1717000120000,
      "totalCount": 3,
      "correctCount": 3
    }
  ]
}
```

---

## 5. 用户配置（config）

可选。

| 字段 | 类型 | 说明 |
|------|------|------|
| `darkMode` | boolean | 深色模式，默认 `false` |
| `randomOptionOrder` | boolean | 刷题时随机打乱选项顺序 |
| `multiBankTypeOrder` | string[] | 多题库联合刷题时题型出现顺序 |

```json
{
  "config": {
    "darkMode": false,
    "randomOptionOrder": false,
    "multiBankTypeOrder": ["judge", "single", "multiple", "fill", "short"]
  }
}
```

## 最小导入示例

只导入一个带 3 道题的题库：

```json
{
  "banks": [
    {
      "id": "demo001",
      "name": "示例题库",
      "questions": [
        {
          "id": "q1",
          "type": "single",
          "content": "1 + 1 = ?",
          "options": [
            { "key": "A", "content": "1" },
            { "key": "B", "content": "2" },
            { "key": "C", "content": "3" }
          ],
          "correctAnswer": ["B"],
          "analysis": ""
        },
        {
          "id": "q2",
          "type": "judge",
          "content": "地球是圆的。",
          "options": [
            { "key": "A", "content": "正确" },
            { "key": "B", "content": "错误" }
          ],
          "correctAnswer": ["A"],
          "analysis": ""
        },
        {
          "id": "q3",
          "type": "fill",
          "content": "中国的首都是____。",
          "options": [],
          "correctAnswer": ["北京"],
          "analysis": ""
        }
      ]
    }
  ]
}
```

---

## 注意事项

1. **`id` 必须唯一**：题目 ID 和题库 ID 不可重复，建议用 `时间戳 + 随机数` 或 UUID
2. **`correctAnswer` 是数组**：即使是单选题也写成 `["B"]`，不要写成 `"B"`
3. **简答评判是模糊匹配**：忽略标点符号、空格、大小写，不需要一字不差
4. **导入会覆盖现有数据**：导入前请备份
5. **选项 `key`**：建议用 A/B/C/D，也支持 1/2/3/4 等
